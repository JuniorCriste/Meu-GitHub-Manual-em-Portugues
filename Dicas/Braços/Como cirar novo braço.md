Comando para criar novo braço:

<pre>$ git push origin nome_do_novo_braco</pre>


Comando para verificar braços existentes:

<pre>$ git branch -a</pre>


Comando enviar atualização no novo braço:

<pre>$ git push origin nome_do_novo_braco</pre>
